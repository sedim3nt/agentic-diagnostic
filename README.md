# Agentic Diagnostic

Are your AI agents actually working? A diagnostic quiz that analyzes your agent setup and gives you a readiness score.

**Live:** [diagnostic.spirittree.dev](https://diagnostic.spirittree.dev)
**Stack:** Next.js, TailwindCSS, Recharts, Framer Motion, OpenRouter
**Status:** Active

## What This Is

Agentic Diagnostic is an interactive assessment tool that evaluates how well someone's AI agent setup is performing. Users answer a series of diagnostic questions about their agent infrastructure, workflows, and outcomes. The responses are then analyzed by AI to generate a personalized readiness report with scores, visualizations, and recommendations.

It's a lead-gen tool for AgentOrchard — help people understand where their agent setup is weak, then offer products that fill those gaps.

## Features

- 📋 **Diagnostic Quiz** — multi-step questionnaire about agent infrastructure
- 🤖 **AI Analysis** — personalized readiness report generated by AI
- 📊 **Score Visualization** — Recharts-powered radar and bar charts
- ✨ **Animated Flow** — smooth Framer Motion transitions between questions
- 🔄 **Loading States** — scanning animation during AI analysis
- 📱 **Responsive** — works across all devices

## AI Integration

**Diagnostic Analyzer** — powered by OpenRouter, analyzes quiz responses to generate a personalized agent readiness report with scores across multiple dimensions (orchestration, memory, tooling, monitoring, etc.) and specific recommendations.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** TailwindCSS
- **Charts:** Recharts
- **Animation:** Framer Motion
- **Database:** None
- **AI:** OpenRouter (via Vercel AI SDK)
- **Hosting:** Vercel

| Variable | Description |
|----------|-------------|
